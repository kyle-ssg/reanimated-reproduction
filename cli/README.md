# Usage
<!-- usage -->
```sh-session
$ npm install -g ssg
$ ssg COMMAND
running command...
$ ssg (-v|--version|version)
ssg/0.0.0 darwin-x64 node-v11.14.0
$ ssg --help [COMMAND]
USAGE
  $ ssg COMMAND
...
```
